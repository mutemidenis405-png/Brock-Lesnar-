Monster Study v2.4

Monster Study is an AI-powered study and revision app designed to make learning smarter, faster, and more interactive. It supports offline and online tutoring, revision tools, and interactive learning activities.


---

Features

Core Features

Offline AI tutor with voice interaction

Document reading and analysis

Highlighting via font color changes

Note-taking and dynamic summaries

Mind maps for visual learning

Cloze exercises, MCQs, and recaps

Library with topic/course classification

Smart board for AI tutoring with override options


Revision & Study Helpers

Automatic retrieval of past papers

Timetable and course outline integration

Interphase breaks and focus management

Dynamic question answering and clarification

Online collaboration: group study and AI-assisted sessions

Video recommendations per topic/subtopic (diverse, curated)

Silent text questions for AI response


Advanced AI Features

Dynamic language switching and multilingual voice support

Memory of past interactions

Smart selection of participants during online sessions

Tutor can search resources online to provide extra context

Installation

1. Clone this repository:



git clone https://github.com/<your-username>/MonsterStudy_v2_4.git

2. Open the project in Android Studio


3. Ensure the Gradle wrapper is present (gradlew)


4. Build the project using:



./gradlew assembleDebug

5. Install the generated APK on your Android device
